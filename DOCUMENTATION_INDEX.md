1. Review [QUICK_START.md](QUICK_START.md) for local setup

2. Review [QUICK_START.md](QUICK_START.md) for setup
3. Verify imports in [RESTRUCTURE_SUMMARY.md](RESTRUCTURE_SUMMARY.md)
